# 在阿里云服务器上安装 nacos

环境：jdk1.8.0，maven-3.8.1

1. 安装 nacos 的官方文档进行安装 nacos
2. 使用 sh startup.sh -m standalone 启动 nacos 后，在本地访问 http://ip:8848/nacos ，其中 ip 就是自己云服务器的公网 ip，然后 8848 是端口号，记得在阿里云服务器安全组中将 8848 端口暴露出来。第一次访问的时候可能会需要输入账号密码，如果没有更改配置的话默认的账号密码都是 nacos。登录完成会就可以看到如下页面了。

这样 nacos 服务就启动成功了。
