# 1. 残差块

左边为 BasicBlock， 右边为 Bottleneck。

使用 involution 替换 ResNet bottleneck block 中的 3×33\times33×3 （使用 3×33\times33×3 或者 7×77\times77×7 的内卷替换）得到了一族新的骨干网络 RedNet，性能和效率优于 ResNet 和其他 self-attention 做 op 的 SOTA 模型。

# 2. 可以进一步提升的地方

  1. 关于广义的 involution 中 kernel 生成函数空间进一步的探索；

  2. 类似于 deformable convolution 加入 offest 生成函数，使得这个 op 空间建模能力的灵活性进一步提升；

  3. 结合 NAS 的技术搜索 convolution-involution 混合结构（原文 Section 4.3）；

  4. 我们在上文论述了 self-attention 只是一种表达形式，但希望 (self-) attention 机制能够启发我们设计更好的视觉模型，类似地 detection 领域最近不少好的工作，也从 DETR 的架构中获益匪浅。
