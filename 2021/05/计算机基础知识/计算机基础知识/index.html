# дёҖгҖҒи®Ўз®—жңәзҪ‘з»ң

# 1.1.TCP/IP еҲҶеұӮ

  • еә”з”ЁеұӮпјҡеә”з”ЁеұӮеҶіе®ҡдәҶеҗ‘з”ЁжҲ·жҸҗдҫӣеә”з”ЁжңҚеҠЎж—¶йҖҡдҝЎзҡ„жҙ»еҠЁгҖӮпјҲDNSгҖҒHTTPпјү
  • дј иҫ“еұӮпјҡеҜ№дёҠеұӮеә”з”ЁеұӮжҸҗдҫӣеӨ„дәҺзҪ‘з»ңиҝһжҺҘдёӯзҡ„дёӨеҸ°и®Ўз®—жңәд№Ӣй—ҙзҡ„ж•°жҚ®дј иҫ“гҖӮеӨ„зҗҶйңҖиҰҒдј иҫ“зҡ„ж•°жҚ®гҖӮ
  • зҪ‘з»ңеұӮпјҡеӨ„зҗҶеңЁзҪ‘з»ңдёҠжөҒеҠЁзҡ„ж•°жҚ®еҢ…пјҢж•°жҚ®еҢ…жҳҜзҪ‘з»ңдј иҫ“зҡ„жңҖе°Ҹж•°жҚ®еҚ•дҪҚпјҢиҜҘеұӮи§„е®ҡдәҶйҖҡиҝҮжҖҺж ·зҡ„и·Ҝеҫ„еҲ°иҫҫеҜ№ж–№и®Ўз®—жңәгҖӮ
  • й“ҫи·ҜеұӮпјҡз”ЁжқҘеӨ„зҗҶиҝһжҺҘзҪ‘з»ңзҡ„зЎ¬д»¶йғЁеҲҶгҖӮ

# 1.2. TCP дёүж¬ЎжҸЎжүӢе’Ңеӣӣж¬ЎжҢҘжүӢ

дёүж¬ЎжҸЎжүӢзҡ„еҺҹеӣ пјҹ

з¬¬дёүж¬ЎжҸЎжүӢжҳҜдёәдәҶйҳІжӯўеӨұж•Ҳзҡ„иҝһжҺҘиҜ·жұӮеҲ°иҫҫжңҚеҠЎеҷЁпјҢи®©жңҚеҠЎеҷЁй”ҷиҜҜжү“ејҖиҝһжҺҘгҖӮ

е®ўжҲ·з«ҜеҸ‘йҖҒзҡ„иҝһжҺҘиҜ·жұӮеҰӮжһңеңЁзҪ‘з»ңдёӯж»һз•ҷпјҢйӮЈд№Ҳе°ұдјҡйҡ”еҫҲй•ҝдёҖж®өж—¶й—ҙжүҚиғҪж”¶еҲ°жңҚеҠЎеҷЁз«ҜеҸ‘еӣһзҡ„иҝһжҺҘзЎ®и®ӨгҖӮе®ўжҲ·з«Ҝзӯүеҫ…дёҖдёӘи¶…ж—¶йҮҚдј ж—¶й—ҙд№ӢеҗҺпјҢе°ұдјҡйҮҚж–°иҜ·жұӮиҝһжҺҘгҖӮдҪҶжҳҜиҝҷдёӘж»һз•ҷзҡ„иҝһжҺҘиҜ·жұӮжңҖеҗҺиҝҳжҳҜдјҡеҲ°иҫҫжңҚеҠЎеҷЁпјҢеҰӮжһңдёҚиҝӣиЎҢдёүж¬ЎжҸЎжүӢпјҢйӮЈд№ҲжңҚеҠЎеҷЁе°ұдјҡжү“ејҖдёӨдёӘиҝһжҺҘгҖӮеҰӮжһңжңүз¬¬дёүж¬ЎжҸЎжүӢпјҢе®ўжҲ·з«ҜдјҡеҝҪз•ҘжңҚеҠЎеҷЁд№ӢеҗҺеҸ‘йҖҒзҡ„еҜ№ж»һз•ҷиҝһжҺҘиҜ·жұӮзҡ„иҝһжҺҘзЎ®и®ӨпјҢдёҚиҝӣиЎҢз¬¬дёүж¬ЎжҸЎжүӢпјҢеӣ жӯӨе°ұдёҚдјҡеҶҚж¬Ўжү“ејҖиҝһжҺҘгҖӮ

д»ҘдёӢжҸҸиҝ°дёҚи®Ёи®әеәҸеҸ·е’ҢзЎ®и®ӨеҸ·пјҢеӣ дёәеәҸеҸ·е’ҢзЎ®и®ӨеҸ·зҡ„и§„еҲҷжҜ”иҫғз®ҖеҚ•гҖӮе№¶дё”дёҚи®Ёи®ә ACKпјҢеӣ дёә ACK еңЁиҝһжҺҘе»әз«Ӣд№ӢеҗҺйғҪдёә 1гҖӮ

  • A еҸ‘йҖҒиҝһжҺҘйҮҠж”ҫжҠҘж–ҮпјҢFIN=1гҖӮ
  • B ж”¶еҲ°д№ӢеҗҺеҸ‘еҮәзЎ®и®ӨпјҢжӯӨж—¶ TCP еұһдәҺеҚҠе…ій—ӯзҠ¶жҖҒпјҢB иғҪеҗ‘ A еҸ‘йҖҒж•°жҚ®дҪҶжҳҜ A дёҚиғҪеҗ‘ B еҸ‘йҖҒж•°жҚ®гҖӮ
  • еҪ“ B дёҚеҶҚйңҖиҰҒиҝһжҺҘж—¶пјҢеҸ‘йҖҒиҝһжҺҘйҮҠж”ҫжҠҘж–ҮпјҢFIN=1гҖӮ
  • A ж”¶еҲ°еҗҺеҸ‘еҮәзЎ®и®ӨпјҢиҝӣе…Ҙ TIME-WAIT зҠ¶жҖҒпјҢзӯүеҫ… 2 MSLпјҲжңҖеӨ§жҠҘж–Үеӯҳжҙ»ж—¶й—ҙпјүеҗҺйҮҠж”ҫиҝһжҺҘгҖӮ
  • B ж”¶еҲ° A зҡ„зЎ®и®ӨеҗҺйҮҠж”ҫиҝһжҺҘгҖӮ

еӣӣж¬ЎжҢҘжүӢзҡ„еҺҹеӣ 

е®ўжҲ·з«ҜеҸ‘йҖҒдәҶ FIN иҝһжҺҘйҮҠж”ҫжҠҘж–Үд№ӢеҗҺпјҢжңҚеҠЎеҷЁж”¶еҲ°дәҶиҝҷдёӘжҠҘж–ҮпјҢе°ұиҝӣе…ҘдәҶ CLOSE-WAIT зҠ¶жҖҒгҖӮиҝҷдёӘзҠ¶жҖҒжҳҜдёәдәҶи®©жңҚеҠЎеҷЁз«ҜеҸ‘йҖҒиҝҳжңӘдј йҖҒе®ҢжҜ•зҡ„ж•°жҚ®пјҢдј йҖҒе®ҢжҜ•д№ӢеҗҺпјҢжңҚеҠЎеҷЁдјҡеҸ‘йҖҒ FIN иҝһжҺҘйҮҠж”ҫжҠҘж–ҮгҖӮ

TIME_WAIT

е®ўжҲ·з«ҜжҺҘж”¶еҲ°жңҚеҠЎеҷЁз«Ҝзҡ„ FIN жҠҘж–ҮеҗҺиҝӣе…ҘжӯӨзҠ¶жҖҒпјҢжӯӨж—¶е№¶дёҚжҳҜзӣҙжҺҘиҝӣе…Ҙ CLOSED зҠ¶жҖҒпјҢиҝҳйңҖиҰҒзӯүеҫ…дёҖдёӘж—¶й—ҙи®Ўж—¶еҷЁи®ҫзҪ®зҡ„ж—¶й—ҙ 2MSLгҖӮиҝҷд№ҲеҒҡжңүдёӨдёӘзҗҶз”ұпјҡ

  • зЎ®дҝқжңҖеҗҺдёҖдёӘзЎ®и®ӨжҠҘж–ҮиғҪеӨҹеҲ°иҫҫгҖӮеҰӮжһң B жІЎж”¶еҲ° A еҸ‘йҖҒжқҘзҡ„зЎ®и®ӨжҠҘж–ҮпјҢйӮЈд№Ҳе°ұдјҡйҮҚж–°еҸ‘йҖҒиҝһжҺҘйҮҠж”ҫиҜ·жұӮжҠҘж–ҮпјҢA зӯүеҫ…дёҖж®өж—¶й—ҙе°ұжҳҜдёәдәҶеӨ„зҗҶиҝҷз§Қжғ…еҶөзҡ„еҸ‘з”ҹгҖӮ
  • зӯүеҫ…дёҖж®өж—¶й—ҙжҳҜдёәдәҶи®©жң¬иҝһжҺҘжҢҒз»ӯж—¶й—ҙеҶ…жүҖдә§з”ҹзҡ„жүҖжңүжҠҘж–ҮйғҪд»ҺзҪ‘з»ңдёӯж¶ҲеӨұпјҢдҪҝеҫ—дёӢдёҖдёӘж–°зҡ„иҝһжҺҘдёҚдјҡеҮәзҺ°ж—§зҡ„иҝһжҺҘиҜ·жұӮжҠҘж–ҮгҖӮ

# 1.3.TCP/UDP еҢәеҲ«

зұ»еһӢйқўеҗ‘иҝһжҺҘдј иҫ“еҸҜйқ жҖ§дј иҫ“еҪўејҸдј иҫ“ж•ҲзҺҮжүҖйңҖиө„жәҗеә”з”ЁеңәжҷҜйҰ–йғЁеӯ—иҠӮ
TCPжҳҜеҸҜйқ еӯ—иҠӮжөҒж…ўеӨҡиҰҒжұӮйҖҡдҝЎж•°жҚ®еҸҜйқ 20-60
UDPеҗҰдёҚеҸҜйқ ж•°жҚ®жҠҘж–Үж®өеҝ«е°‘иҰҒжұӮйҖҡдҝЎйҖҹеәҰеҝ«8 дёӘеӯ—иҠӮ

# 1.4. иҫ“е…Ҙ url еҗҺеҲ°жҳҫзӨәдё»йЎөиҝҮзЁӢ

  1. жөҸи§ҲеҷЁйҖҡиҝҮ DNS жҹҘжүҫеҹҹеҗҚжүҖеҜ№еә”зҡ„ IP ең°еқҖпјӣ
  2. жөҸи§ҲеҷЁдёҺзӣ®ж ҮжңҚеҠЎеҷЁйҖҡиҝҮдёүж¬ЎжҸЎжүӢе»әз«Ӣ TCP иҝһжҺҘпјӣ
  3. жөҸи§ҲеҷЁйҖҡиҝҮ http еҚҸи®®еҸ‘йҖҒиҜ·жұӮпјӣ
  4. жңҚеҠЎз«Ҝе“Қеә” http иҜ·жұӮпјӣ
  5. жөҸи§ҲеҷЁеҸ‘йҖҒиҜ·жұӮиҺ·еҸ– html дёӯзҡ„иө„жәҗз„¶еҗҺжёІжҹ“еұ•зӨәеҮәжқҘгҖӮ
  6. йҮҠж”ҫиҝһжҺҘгҖӮ

# 1.5. зҠ¶жҖҒз Ғ

зҠ¶жҖҒз Ғзұ»еҲ«еҺҹеӣ 
1XXдҝЎжҒҜзұ»зҠ¶жҖҒз ҒжҺҘж”¶зҡ„иҜ·жұӮжӯЈеңЁеӨ„зҗҶ
200 OKжҲҗеҠҹзҠ¶жҖҒз ҒиҜ·жұӮиў«жӯЈеёёеӨ„зҗҶ
204 No ContentжңҚеҠЎеҷЁжҲҗеҠҹеӨ„зҗҶиҜ·жұӮпјҢдҪҶжҳҜдёҚдјҡиҝ”еӣһиө„жәҗ
206 Partial Contentе®ўжҲ·з«ҜиҝӣиЎҢдәҶиҢғеӣҙиҜ·жұӮпјҢе№¶дё”жңҚеҠЎеҷЁжҲҗеҠҹжү§иЎҢиҝҷйғЁеҲҶ Get иҜ·жұӮ
301 Moved PermanentlyйҮҚе®ҡеҗ‘зҠ¶жҖҒз ҒиҜҘзҠ¶жҖҒз ҒиЎЁзӨәиҜ·жұӮзҡ„иө„жәҗе·Іиў«еҲҶй…ҚдәҶж–°зҡ„ URIпјҢд»ҘеҗҺеә”дҪҝз”Ёиө„жәҗзҺ°еңЁжүҖжҢҮзҡ„ URI
302 FoundиЎЁзӨәиҜ·жұӮзҡ„иө„жәҗе·Іиў«еҲҶй…Қж–°зҡ„ URIпјҢеёҢжңӣз”ЁжҲ·жң¬ж¬ЎиғҪдҪҝз”Ёж–°зҡ„ URI и®ҝй—®
303 See Otherз”ұдәҺиҜ·жұӮеҜ№еә”зҡ„иө„жәҗеӯҳеңЁеҸҰдёҖдёӘ URIпјҢеә”дҪҝз”Ё GET ж–№жі•е®ҡеҗ‘иҺ·еҸ–иҜ·жұӮзҡ„иө„жәҗ
304 Not Modifiedиө„жәҗе·ІжүҫеҲ°пјҢдҪҶжңӘз¬ҰеҗҲжқЎд»¶иҜ·жұӮ
400 Bad Requestе®ўжҲ·з«Ҝй”ҷиҜҜзҠ¶жҖҒз ҒиҜҘзҠ¶жҖҒз ҒиЎЁзӨәиҜ·жұӮжҠҘж–ҮдёӯеӯҳеңЁиҜӯжі•й”ҷиҜҜ
401 Unauthorizedз¬¬дёҖж¬ЎиЎЁзӨәйңҖиҰҒи®ӨиҜҒпјҢз¬¬дәҢж¬ЎиЎЁзӨәи®ӨиҜҒеӨұиҙҘ
403 ForbiddenиЎЁзӨәиҜ·жұӮзҡ„иө„жәҗиў«жңҚеҠЎеҷЁжӢ’з»қ
404 Not FoundжңҚеҠЎеҷЁдёҠжІЎжңүиҜ·жұӮзҡ„иө„жәҗ
500 Internal Server ErrorжңҚеҠЎеҷЁй”ҷиҜҜзҠ¶жҖҒз ҒиЎЁзӨәжңҚеҠЎеҷЁжү§иЎҢиҜ·жұӮж—¶еҸ‘з”ҹй”ҷиҜҜ
503 Service UnavailableиЎЁжҳҺжңҚеҠЎеҷЁжҡӮж—¶еӨ„дәҺи¶…иҙҹиҪҪжҲ–жӯЈеңЁеҒңжңәз»ҙжҠӨпјҢж— жі•еӨ„зҗҶиҜ·жұӮ

301пјҢ302пјҢ303 иҷҪз„¶и§„е®ҡдёҚжҳҜиҝҷж ·пјҢдҪҶжҳҜе®һйҷ…дёҠжөҸи§ҲеҷЁеңЁе“Қеә”зҠ¶жҖҒз Ғиҝ”еӣһзҡ„ж—¶еҖҷйғҪдјҡе°Ҷ POST ж”№жҲҗ GETгҖӮ

# 1.6.Http1.0,1.1,2.0

# Http1.0гҖҒ1.1

  • зј“еӯҳзӯ–з•Ҙпјҡhttp1.0 зҡ„зј“еӯҳзӯ–з•Ҙдё»иҰҒжҳҜдҫқиө– header дёӯзҡ„ If-Modiified-SinceпјҢExpire (еҲ°жңҹ)пјҢhttp1.1 зҡ„зј“еӯҳзӯ–з•ҘиҰҒжҜ” http1.0 з•ҘеӨҡпјҢдҫӢеҰӮ Entity tag (е®һдҪ“ж Үзӯҫ)пјҢIf-Unmodified-SinceпјҢIf-MatchпјҢIf-None-Match зӯү.

  • е®ҪеёҰзҪ‘з»ңиҝһжҺҘдјҳеҢ–пјҡhttp1.0 дёӯдјҡеӯҳеңЁдёҖдәӣжҖ§иғҪжөӘиҙ№пјҢжҜ”еҰӮеҸӘйңҖиҰҒеҜ№иұЎдёӯзҡ„дёҖйғЁеҲҶпјҢдҪҶжҳҜжҜҸж¬ЎиҜ·жұӮиҝ”еӣһзҡ„еҚҙжҳҜж•ҙдёӘеҜ№иұЎпјҢиҝҷж— з–‘йҖ жҲҗдәҶжҖ§иғҪзҡ„жҚҹе®ігҖӮhttp1.1 еҲҷдёҚз„¶пјҢе®ғеҸҜд»ҘйҖҡиҝҮеңЁиҜ·жұӮеӨҙеӨ„и®ҫзҪ® range еӨҙеҹҹпјҢе°ұеҸҜд»Ҙиҝ”еӣһиҜ·жұӮиө„жәҗзҡ„жҹҗдёҖйғЁеҲҶпјҢд№ҹе°ұжҳҜиҝ”еӣһз Ғдёә 206 (Partial Content) зҡ„ж—¶еҖҷпјҢиҝҷеҜ№дәҺжҖ§иғҪдјҳеҢ–еҫҲжңүеҝ…иҰҒгҖӮ

  • ж–°еўһйғЁеҲҶй”ҷиҜҜйҖҡзҹҘпјҡHttp1.1 ж–°еўһдәҶ 24 дёӘй”ҷиҜҜзҠ¶жҖҒе“Қеә”з Ғпјӣ

  • Host еӨҙеӨ„зҗҶпјҡhttp1.1 еңЁиҜ·жұӮж¶ҲжҒҜе’Ңе“Қеә”ж¶ҲжҒҜдёӯеҠ е…Ҙ HOST еӨҙеҹҹпјҢз”ЁжқҘдј йҖ’дё»жңәеҗҚпјӣ

  • й•ҝиҝһжҺҘпјҡhttp1.1 ж”ҜжҢҒй•ҝиҝһжҺҘгҖӮ

# Http2.0гҖҒ1.x

  1. http1 зҡ„и§ЈжһҗжҳҜеҹәдәҺж–Үжң¬еҚҸи®®зҡ„еҗ„ејҸи§ЈжһҗпјҢиҖҢ http2.0 зҡ„еҚҸи®®и§ЈжһҗжҳҜдәҢиҝӣеҲ¶ж јејҸпјҢжӣҙеҠ зҡ„ејәеӨ§

  2. еӨҡи·ҜеӨҚз”Ё (Mutiplexing) : дёҖдёӘиҝһжҺҘдёҠеҸҜд»ҘжңүеӨҡдёӘ request, дё”еҸҜд»ҘйҡҸжңәзҡ„ж··еңЁдёҖиө·пјҢжҜҸдёӘдёҚеҗҢзҡ„ request йғҪжңүеҜ№еә”зҡ„ id, жңҚеҠЎз«ҜеҸҜд»ҘйҖҡиҝҮ request_id жқҘиҫЁеҲ«пјҢеӨ§еӨ§еҠ еҝ«дәҶдј иҫ“йҖҹзҺҮ

  3. header еҺӢзј©: http1.x дёӯзҡ„ header йңҖиҰҒжҗәеёҰеӨ§йҮҸдҝЎжҒҜгҖӮиҖҢдё”жҜҸж¬ЎйғҪиҰҒйҮҚеӨҚеҸ‘йҖҒ.http2.0 дҪҝз”Ё encode жқҘеҮҸе°‘дј иҫ“зҡ„ header еӨ§е°ҸгҖӮиҖҢдё”е®ўжҲ·з«Ҝе’ҢжңҚеҠЎз«ҜеҸҜд»Ҙеҗ„иҮӘзј“еӯҳ (cache) дёҖд»Ҫ header filed иЎЁпјҢйҒҝе…ҚдәҶ header зҡ„йҮҚеӨҚдј иҫ“пјҢиҝҳеҸҜд»ҘеҮҸе°‘дј иҫ“зҡ„еӨ§е°Ҹ.
