zookeeper的简单实战

发表于 2021-05-12 分类于 zookeeper学习笔记

1. 安装与使用

1.1. 安装

下载镜像

docker pull zookeeper

运行 docker

docker run -d --name zookeeper -p 2181:2181 zookeeper

1.2. 连接 zookeeper 服务器

  1. 进入容器 docker exec -it ContainerID /bin/bash
  2. 切换到 bin 目录，并连接 ./zkCli.sh -server 127.0.0.1:2181
