一、管理员前端

简介

管理员前端系统是根据 vue-admin-template 前端模板来进行搭建的。

使用

首先编写 api 对应后端的 controller 请求，然后在 router.js 中编写路由，最后在 views 中编写展示数据的页面。在 utils/request.js 中编写好了如何向后端请求的 post 函数。
