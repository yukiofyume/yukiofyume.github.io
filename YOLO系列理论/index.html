# 通道数的含义

  1. class scores 代表的是这个像素点对应于原图片的网格 20 类对象的概率。之所以 20 层是因为网络将每个类别的概率都记录下来了，就是说每一层的每一个像素点对应的是类别 C1、C2、...、C20C1、C2、...、C20C1、C2、...、C20 的概率分别是多少。

  2. 置信度

    confidence=Pr(Object)∗IOUpregtconfidence = Pr(Object) * IOU{^{gt}_{pre}}confidence=Pr(Object)∗IOUpregt​

    公式右边 = 次网格中含有目标的概率 * 预测框和真实框之间的交并比。（注意 Pr(Object)Pr(Object)Pr(Object) 指的是含不含有目标，而不是含不含有某个类别）

  3. box：4 个值，分别问预测框中心点坐标以及框的高和宽

# Top-1

假设有十个分类，现在有十张测试图片，那么一张图片输入到网络中，就可以得到十个分类概率。Top-1是这十个类别概率中取概率最高的一个类别，如果此测试图片的类别为概率最高类别，则表明预测成功，反正预测错误。
