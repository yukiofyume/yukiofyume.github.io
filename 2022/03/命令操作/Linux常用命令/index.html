Linux常用命令

发表于 2022-03-01 分类于 命令操作

1.rm 命令

1.删除任何 .log 文件，删除前逐一询问

rm -i *.log

2.mv 命令

使用这个命令重命名的时候可以加上 mv -i 1.txt 2.txt ，这样如果是存在，就会提示是否需要覆盖

3.cp 命令

同样可以 cp -i ，来提示

4.cat 命令

  1. 一次性显示整个文件：cat filename
  2. 新建一个文件：cat > filename
  3. 合并几个文件：cat file1 file2 > file
  4. 反向输出文件：tac filename
  • -n 输出所有行号
  • -b 对非空输出行号

5.more 命令

功能类似于 cat, more 会以一页一页的显示方便使用者逐页阅读，而最基本的指令就是按空白键（space）就往下一页显示，按 b 键就会往回（back）一页显示

  • +n 从笫 n 行开始显示
  • -n 定义屏幕大小为n行
  • +/pattern 在每个档案显示前搜寻该字串（pattern），然后从该字串前两行之后开始显示
  • -c 从顶部清屏，然后显示
  • -d 提示“Press space to continue，’q’ to quit（按空格键继续，按q键退出）”，禁用响铃功能
  • -l 忽略Ctrl+l（换页）字符
  • -p 通过清除窗口而不是滚屏来对文件进行换页，与-c选项相似
  • -s 把连续的多个空行显示为一行
  • -u 把文件内容中的下画线去掉

6.head 命令

打印前面几行

  1. 显示前20行：head -n 20 a.txt
  2. 显示前20字节：head -c 20 a.txt
  3. 从倒数10行开始显示：hean -n -10 a.txt

7.tail 命令

-f 循环读取（常用于查看递增的日志文件） tail -f filename

8.查找文件命令

命令 作用
which 查找可执行文件的位置
whereis 查看文件的位置
locate 配合数据库查看文件位置
find 实时搜寻磁盘，查询文件名称

8.1.which

查看命令是否存在 which ls

8.2.whereis

whereis 命令只能用于程序名的搜索。只搜索二进制文件（参数-b）、man说明文件（参数-m）和源代码文件（参数-s）。如果省略参数，则返回所有信息。whereis 及 locate 都是基于系统内建的数据库进行搜索，因此效率很高，而find则是遍历硬盘查找文件。

8.3.locate

基于系统内部数据库，查找很快。

  1. 查找和 pwd 相关的所有文件 ：locate pwd

8.4.find

  1. 查找 48 小时内修改过的文件：find -atime -2
  2. 查找当前目录以 .txt 结尾的文件：find ./ -name '*.txt'
  3. 查找大于 1k 的文件：find -size +1000c

9.ps 命令

ps(process status)，用来查看当前运行的进程状态，一次性查看，如果需要动态连续结果使用 top

11.查看端口占用情况

11.1.lsof

lsof(list open files)是一个列出当前系统打开文件的工具。

lsof -i:port

11.2.netstat

netstat -tunlp|grep port
