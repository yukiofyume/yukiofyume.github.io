1.1 版本穿梭

  • –soft：仅仅在本地库移动 head 指针
  • –mixed：在本地库、重置暂存区移动 head 指针
  • –soft：在本地库、重置暂存区、重置工作区移动 head 指针

回退到第二个版本

git reset --hard 5bbcdc1
