# IOU 的发展及意义

边界框回归是目标检测的关键步骤。

边界框回归算法选择的好坏与否会影响到最终的预测结果，下图中，即使正确的分类出飞机，但是 IOU 太小了，也说明没有正确地检测出飞机，此时如果能够对红色的框进行微调，那么定位就会更加准确，所以说边界框回归就是来微调这个窗口的。

下图中，红色框代表先验框，绿色框代表真实框，蓝色框代表通过边界框回归算法得出的预测目标框。

很明显，从概念上来说，IOU 越大就说明预测的效果越好，越接近真实框。

从几何上来说，影响边界框回归主要有三个因素，即重叠面积、中心点距离和纵横比，在此基础上提出了 CIOU_Loss（Complete IoU） ，从而可以使得模型收敛的更快、取得更好的效果。

IOU 的发展过程：Smooth L1 Loss-> IoU Loss（2016）-> GIoU Loss（2019）-> DIoU Loss（2020）->CIoU Loss（2020）

最基本的 IOU 存在一定的问题，

  1. 当预测框和目标框不相交时，IOU=0，此时损失函数不可导

  2. 当预测框大小相同的时候，两个 IOU 也相同的时候，此时 IOU_loss 无法评估两者相交情况的不同

    红色为预测框，黑色为目标框。

为了解决 IOU loss 所存在的问题，与 2019 年提出了 GIOU_Loss

可以看到 GIOU_Loss 是在 IOU_Loss 的基础上加入了一个惩罚因子，主要是为了在预测框和目标框不相交情况下损失函数不可导导致的无法训练问题。

但是 GIOU_Loss 也存在一定的问题：

如上图所示，当预测框完全位于目标框内部且预测框大小相同的时候，这三个图的 GIOU 是相同的，也就是说 GIOU 的惩罚因子恒为 0，此时 GIOU 就退化成了 IOU，无法区分预测框和目标框的相对位置关系。

当时在如下场景下，DIOU 也会存在一些问题。

# CIOU_Loss

CIOU 就是为了在预测框和目标框重叠的时候，如何使得模型回归的更加准确。

通过统一坐标，IOU_Loss 考虑了重叠面积问题，DIOU 在考虑了重叠面积以及中心点距离问题，而 CIOU_Loss 则同时考虑了重叠面积、中心点距离以及纵横比问题。

LOSS问题
IOU_Loss主要考虑预测框和目标框重叠面积
